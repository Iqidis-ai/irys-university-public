# Irys University

Irys University is a public research and education library about programmable data, AI infrastructure, agent workflows, and the systems-level problems that appear when data has to be stored, verified, executed, reused, and trusted.

Irys is the programmable datachain: a data layer that brings storage, execution, and verification into one environment.

## About Irys

Irys is built for systems that rely on data. Modern AI and agent workflows need data that can stay available, verifiable, structured, and usable after it moves through many transformations. Irys treats data as a programmable object rather than passive storage.
